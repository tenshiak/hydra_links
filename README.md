To be used with Hydra Launcher. Magnet links only, except AnkerGames.

Trackers:
- **Combined** - Grabs magnet links from several sources and combines them into one file. Duplicate magnets removed. Order of importance:  FitGirl (official site) -> 1337x (FitGirl, DODI, KaOs) -> Rutor -> RuTracker. RuTracker doesn't include files marked as Unverified or Doubtful.
- **AnkerGames** - Only informational, there are no links inside. Detects if there is a new upload available, but it doesn't download automatically with Hydra. Not included in the Combined file.
- **TPB dauphong** - Many people swear by him, but his uploads include IGG-GAMES uploads, and they are considered not trustworthy. Use at your own risk. Not included in the Combined file.
